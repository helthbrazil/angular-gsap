# GsapTest

This project demonstrates the use of GSAP animations in Angular.

Live demo:\
https://helthbrazil.github.io/angular-gsap

## How to Run the Project

### Install dependencies

    npm i

### Start the application

    npm start

Access the application at:\
http://localhost:4200/

## Build

To create a production build:

    ng build

The build output will be located in the `dist/` directory.

## Code Scaffolding

Generate components, services, directives and more:

    ng generate component component-name
    ng generate service service-name

## Running Unit Tests

    ng test

## Running End-to-End Tests

    ng e2e

To learn more about Angular CLI, visit the official documentation:\
https://angular.io/cli
